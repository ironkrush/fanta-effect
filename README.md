# fanta-effect

This is a fanta effect which i made by using gsap which is a great animation library

When you scroll down the image of drinking cane and other stuff like orange slice and leaves also move and the cane takes another position where some text is written about the same and when we move it more below then there's different boxes in which we can show multiple drinking flavours

We can make it complex by adding some more animations like cane should roll down or drink into the cane flows like that i will update this design in future
